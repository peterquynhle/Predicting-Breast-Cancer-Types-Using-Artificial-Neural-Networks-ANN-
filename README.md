# Predicting-Breast-Cancer-Types-Using-Artificial-Neural-Networks-ANN:

Data source: 
	- Breast Cancer Wisconsin (Diagnostic) Data Set
	- ftp ftp.cs.wisc.edu cd math-prog/cpo-dataset/machine-learn/WDBC/
